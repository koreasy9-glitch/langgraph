# 바닐라 JS 테트리스

정적 HTML, CSS, JavaScript만으로 동작하는 테트리스 예제입니다. 별도 빌드 과정 없이 `index.html`을 브라우저에서 바로 열면 실행됩니다.

## 특징
- 7-백 랜덤으로 공정하게 블록을 생성합니다.
- 벽킥을 적용한 회전과 고스트 피스로 직관적인 낙하 위치를 제공합니다.
- 라인 수에 따라 자동으로 레벨이 오르며 낙하 속도가 빨라집니다.
- 다음 블록과 홀드 슬롯을 포함한 UI를 제공합니다.

## 조작법
- 좌우 이동: `←` / `→`
- 회전: `↑`
- 소프트 드랍: `↓`
- 하드 드랍: `Space`
- 홀드/교체: `C`

## 실행 방법
1. `index.html`을 더블 클릭하거나, 간단한 HTTP 서버로 열어주세요. 예: `python -m http.server` 후 브라우저에서 `http://localhost:8000/examples/tetris/` 접속
2. "새 게임 시작" 버튼을 눌러 플레이하세요.
